  • Chapter 1: Basics

    This chapter introduces the basic concepts which are necessary to understand the subsequent chapters. We will first focus on the basic concepts of (supervised) machine learning, explain the difference between regression and classification, show how to evaluate and compare Machine Learning models and formalize the concept of learning. The second subchapter will focus in tasks and challenges, as well as frequently used benchmarks, in NLP in order to provide some motivation from the practitioner&rsquo;s perspective.

  • Chapter 2: Deep Learning Basics

    This chapter treats the supervised regression task in more detail. We will see different loss functions for regression, how a linear regression model can be used from a Machine Learning perspective, how to extend it with polynomials for greater flexibility and finally a fundamentally different approach - k-NN regression.
